# F1 Event Booking System

This is a web application developed using Java Spring Boot for booking Formula 1 race tickets.

## Technologies Used
- Java
- Spring Boot
- HTML / CSS / Bootstrap
- Thymeleaf
- GitHub

## Features
- View F1 race calendar
- Book tickets for races
- Admin login
- Add new races
- Delete races
- File-based data storage

## Project Structure
MVC Architecture:
- Model: RaceEvent, Booking
- Controller: HomeController, BookingController, AdminController
- View: HTML templates
